## Creative Notes

This is Reel 1. It focuses on the "stars wanted New York's lights but not New York's smoke" argument.

The visual is intentionally static aside from the waveform bars, so the motion supports the narration without distracting from it. The corrected export uses locked text layers, full-image containment, no image panning, no sliding text, and no intro sound effect.

The final audio is one continuous OpenAI narration pass from the approved script, not stitched article snippets.
